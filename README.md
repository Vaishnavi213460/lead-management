Setup steps : 
-----------------
    1. git clone "repo-url"
    2. cd lead-management
    3. run "composer install"
    4. run "npm install"
    5. run "php artisan key:generate"
    6. run "php artisan migrate"
    7. run "npm run dev" in one terminal
    8. run "php artisan serve" in another terminal

    Use the url in browser register with username and password
